# warp-t01-json

## How to request
### GET method

1. Open web browser
2. Call `http://localhost:3000/expr?q=HelloWorld!`
3. You can find `HelloWorld!` in your web browser

### POST method

`curl -X POST http://localhost:3000/`

### PUT method

`curl -X PUT http://localhost:3000/`
### DELETE method

`curl -X DELETE http://localhost:3000/`
